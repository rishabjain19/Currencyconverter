# Currency Converter

A simple web application that converts currency values using real-time exchange rates.

## Live Demo
https://currencyconverter-lemon.vercel.app/

## Repository
https://github.com/rishabjain19/Currencyconverter

## Features
- Currency conversion in real time
- Supports multiple currencies
- Simple and responsive UI

## Built With
- HTML
- CSS
- JavaScript

## Usage
1. Enter the amount
2. Select source and target currencies
3. View the converted value instantly
